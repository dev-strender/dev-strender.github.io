# 파일 복사할때 symlink를 해제하여 복사하는 방법

AI 데이터셋에서 동일한 이미지가 여러 버젼으로 관리하게 되는 경우, (특히 이미지 파일) 심볼릭 링크로 관리되는 경우가 다소 발생한다.
단순히 `cp` 명령어를 통해 복사를 하게 되면, 심볼릭 링크만 복사가 되는 경우가 있다.

이럴때는, 해제해서 하드카피를 해줘야 하며, 해당 명령어는 `cp -RL {SRC} {TARGET}` 옵션으로 복사해주면 된다.
